Example 4 - Lists of controllers

This is the Example4Controller.

The Section2Controller is a subclass of BindingController which uses RivetJS to give you features like two-way data binding and some other nice tricks.

As with TemplateControllers, with this kind of controllers the template is going to be the view. BindingControllers talk with RivetJS binding an object that has the form:

{model: aModel, controller: self}

where self is the controller that you are using

Take this one for example... the button to confirm is going to enable only when the input has a non empty content. You'll notice the model will perceive the change on blur (RivetJS's default way to notice model change)

If you observe the template you'll see it has some other elements that are going to show/hide depending on the state of the model and controller. Study them to see how you can make your own!
